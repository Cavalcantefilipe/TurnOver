# Test TurnOver

this project was done in Php Laravel and React


## Installation back-end

Configure your .env to your data-base

```bash
cd back-end
```

```bash
composer install
```

```bash
php artisan key:generate
```

```bash
php -S localhost:8000 -t public
```

## Installation front-end


```bash
cd web
```

```bash
yarn start
```
